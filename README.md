# BinaryProject


dizinin sol elemanları incelenir ve  7 ile 5 i karşılaştırılr. 5, 7'den küçük olduğu için sol tarafa yazılır:
            7
           /
          5

Sırada 1 var. 1, 7'den küçük sola, 5'ten küçük sola yerleştirilir:
            7
           /
          5
         /
        1

8, 7'den büyük sağa:
            7
           / \
          5   8
         /
        1

          5   8
         /
        1

          3
